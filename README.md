# Python Selenium을 활용한 sk엔카 데이터 크롤링

## 소개
__동급 매물__ 과 __팔린 매물__ 의 데이터를 받아와 _팔림세_ 를 계산하기 위함<br>
_팔림세_ 는 타겟 차량의 팔림 트렌드를 확인 할 수 있는 수식이고 중고차 매매에서 영향력 있는 지표임<br>
>그 수식은 다음과 같음:<br>
>$$ 일정 기간 동안의 ({동급 매물 수 \over 팔린 매물 수})$$
